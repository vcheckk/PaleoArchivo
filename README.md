# 🦖 PaleoArchivo

**PaleoArchivo** es una enciclopedia interactiva y visual sobre paleontología, centrada en la era de los dinosaurios. El proyecto combina una estética de "cuaderno de campo" con una interfaz moderna y reactiva.

## ✨ Características Principales

* **Fichas Técnicas Detalladas**: Cada dinosaurio cuenta con su ID único, dieta, dimensiones (longitud/altura) y descripción.
* **Diseño "Expedición"**: Paleta de colores inspirada en sedimentos y fósiles.
* **Interactividad**: Animaciones de escala (*hover effects*) y transiciones suaves para una navegación fluida.
* **Totalmente Responsive**: Optimizado para su visualización en dispositivos móviles mediante túneles de desarrollo (Ngrok/Localtunnel).
* **Categorización por Dietas**: Etiquetas dinámicas con colores codificados para carnívoros, herbívoros, etc.

## 🚀 Tecnologías Utilizadas

* **React 18**: Biblioteca principal para la interfaz.
* **Tailwind CSS**: Framework de utilidades para un diseño rápido y consistente.
* **Vite**: Herramienta de construcción (build tool) de última generación.
* **Lucide React**: (Opcional) Iconografía técnica.
